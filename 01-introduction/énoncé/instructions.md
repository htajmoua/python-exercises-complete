# Instructions 

Dans cet exercice, vous allez écrire directement dans un fichier python qui se nomme `main.py`.

## Exercices de base

1. Utilisez la fonction `print` pour afficher la phrase `"Bienvenue dans le monde de la programmation Python !"`
2. Calculez le résultat de `25 + 48 * 3` et affichez le résultat (Vous pouvez placer le calcul directement dans la fonction `print` pour l'afficher)
3. Affichez le résultat du calcul `(120 - 45) / 5`

## Questions supplémentaires

4. Affichez votre prénom sur une ligne, puis votre ville sur une autre ligne (utilisez deux `print` séparés)
5. Calculez et affichez le résultat de `2 ** 10` (2 puissance 10)
6. Affichez le résultat de `100 % 7` (modulo - reste de la division)

## Question bonus

7. Dans un seul `print`, affichez "Python" et "Programmation" séparés par un espace (indice : utilisez une virgule dans le print)
