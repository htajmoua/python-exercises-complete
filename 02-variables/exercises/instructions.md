# Instructions  

## Exercices de base

1. Créez une variable nommée `produit` et attribuez-y le nom d'un produit (par exemple `"Ordinateur"`).
2. Créez une variable nommée `prix` et attribuez-y un prix (par exemple `899`).
3. Affichez une phrase contenant le **produit** et le **prix** en utilisant les variables que vous venez de créer. Par exemple, `"Le produit Ordinateur coûte 899 euros."` (Utilisez le f-string)
4. Modifiez la valeur de la variable `prix` pour appliquer une **réduction de 15%** (multipliez le prix par 0.85).
5. Affichez à nouveau la phrase avec le prix réduit. Par exemple, `"Après réduction, le produit Ordinateur coûte 764.15 euros."` (Utilisez le f-string)

## Questions supplémentaires

6. Créez une variable `quantite` avec la valeur `3` et calculez le **prix total** (prix réduit × quantité).
7. Affichez : `"Pour 3 Ordinateur(s), le total est de X euros."` en utilisant les variables (utilisez le f-string)
8. Créez une variable `devise` avec la valeur `"USD"` et modifiez votre affichage précédent pour utiliser cette devise au lieu de "euros"

## Question bonus

9. Créez une variable `tva` avec la valeur `0.20` (20%) et calculez le prix TTC (prix total + TVA). Affichez le résultat formaté.
